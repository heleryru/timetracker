# Timetracker
This is a simple timetracker app that allows you to track your time spent on projects. It is built with Vue.js and Express.js.

## Project management

[https://www.pivotaltracker.com/n/projects/2632253]

## Project setup

```
cp .env.example .env
```

```
npm install
```

### Running the app

```
npm start
```
